━━━ <b>CLOSED GIANTS</b> 🔒

<b>Gemini 3.5 Flash</b> — GA May 19. Google's flash-tier model now beats Gemini 3.1 Pro on coding and agentic benchmarks (76.2% Terminal-Bench 2.1, 83.6% MCP Atlas) with dynamic thinking enabled by default. $1.50/$9.00 per 1M tokens, 1M context window. <a href="https://llm-stats.com/models/gemini-3.5-flash">→ Stats</a>

<b>GPT-5.5 Instant</b> — Released May 5. OpenAI's new ChatGPT default — lighter and faster than GPT-5.5 rather than a benchmark maximizer. 81.2% AIME 2025, 76% MMMU-Pro multimodal. Replaced GPT-5.3 Instant. <a href="https://llm-stats.com/llm-updates">→ Updates</a>

━━━ <b>PREMIER OPEN</b> 🔓

<b>Mistral Medium 3.5</b> — Released April 29. 128B dense under modified MIT, collapsing Mistral's prior Medium 3.1, Magistral, and Devstral 2 into one model with code, reasoning, and vision. 77.6% SWE-bench Verified, 256K context, self-hostable on 4 GPUs. API at $1.50/$7.50 per 1M tokens. <a href="https://huggingface.co/mistralai/Mistral-Medium-3.5-128B">→ HF</a>

<b>Qwen3.6-27B</b> — Released April 22. Alibaba's dense 27B outscores the prior 397B MoE flagship on SWE-bench Verified (77.2 vs 76.2) and Terminal-Bench (59.3 vs 52.5). Apache 2.0, 262K context (extensible to 1M), with Thinking Preservation retained across conversation turns. <a href="https://huggingface.co/Qwen/Qwen3.6-27B">→ HF</a>

<b>Kimi K2.6</b> — Released April 20. Moonshot AI's 1T-total / 32B-active MoE leads open-weight coding at score 54 on AI Analysis Intelligence Index v4.0. 256K context. Current top open-source pick for agentic engineering tasks. <a href="https://codersera.com/blog/kimi-k2-6-vs-deepseek-v4-vs-glm-5-1-2026/">→ Comparison</a>

<b>DeepSeek V4-Pro</b> — Released April 24. 1.6T total / 49B active MoE with a 1M-token context window. Scores 52 on AI Analysis Index v4.0. Companion V4-Flash (284B / 13B active) offers lower-cost inference from the same family. Open weights on Hugging Face. <a href="https://simonwillison.net/2026/apr/24/deepseek-v4/">→ Overview</a>

<b>ZAYA1-8B</b> — Released May 6. Zyphra's MoE with only 760M active / 8.4B total parameters, trained on AMD Instinct MI300X. Reaches 91.9% AIME'25 with Markovian RSA — matching much larger reasoning models. Apache 2.0, serverless endpoint on Zyphra Cloud. <a href="https://arxiv.org/abs/2605.05365">→ Technical Report</a>

━━━ <b>SPECIALIZED</b> 🎯

<b>SubQ 1M-Preview</b> — Released May 5. Subquadratic's commercial debut: the first LLM on a fully subquadratic sparse-attention architecture where compute scales linearly with context length. Claims 12M-token context at 1,000× lower compute than transformer equivalents at that length. $29M seed, independent benchmarks pending. <a href="https://subq.ai/introducing-subq">→ Announcement</a>
